## 💡 수입/지출 기록하는 가계부

[🔗 Site Link](https://savings-book-frontend.vercel.app/) <br/>

`Tect Stack`
- React
    - redux-toolkit
    - react-router-dom
    - axios
      - interceptor
    - hooks
- styled-component
- apex chart
- localstorage

### 📄 화면 구성
- 로그인/회원가입
  - 닉네임 등록/수정
- 이번 달 수입/지출 총 금액
- 달력에 기록된 날짜 별 수입/지출 금액
- 수입/지출 내역 등록
- 이번 달 목표 소비 금액 설정
- 이번 달 카테고리별 소비 측정 (원형 그래프)
  - apex chart
- 저번 달, 이번 달 일별 소비 측정 (라인 그래프)
  - apex chart
- 목표 달성 뱃지
- 테마 변경 기능
  - localstorage
